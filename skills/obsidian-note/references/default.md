# 通用笔记模板

## YAML 示例

```yaml
created: 2026-03-19
modified: 2026-03-19
title: 笔记标题
url:
author:
description:
tags:
  - 标签
slug: note-title
cover:
```

## 内容结构

```markdown
## 概述

## 正文

## 参考资料
```

## 填写说明

- 当用户未指定笔记类型，或内容不属于其他类型时使用此模板
- `tags` 根据内容主题自由填写
- `description` 一句话概括笔记主旨
- 此模板无 `One more thing—记录` 块（通用笔记不一定与日记关联）
